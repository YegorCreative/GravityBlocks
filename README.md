# GravityBlocks
GravityBlocks is a simple, browser-based falling blocks game inspired by classic Tetris-style gameplay.  
Built with HTML, CSS, and vanilla JavaScript, and deployed using GitHub Pages.

## Features

- Classic falling-blocks gameplay (rotate, move, drop)
- Line clearing, scoring, and level progression
- Game over detection and restart
- Keyboard controls for desktop
- Mobile-friendly layout with on-screen touch controls
- Pure HTML/CSS/JS — no frameworks required

---

## Controls

**Desktop (Keyboard):**

- Left / Right Arrow – Move piece
- Up Arrow – Rotate piece
- Down Arrow – Soft drop
- Space – Hard drop
- P – Pause / Resume
- R – Restart game

**Mobile (Touch):**

- On-screen buttons for:
  - Move left / right
  - Rotate
  - Drop (soft / hard)

*(Exact layout may change as the project evolves.)*
